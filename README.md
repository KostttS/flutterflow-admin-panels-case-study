## Technical challenges

### FlutterFlow Web as an admin tool

FlutterFlow Web was used to build practical admin panels connected to the same backend as the mobile apps.

The challenge was to keep admin flows simple, fast and reliable while working with Firebase or Supabase data structures.

### Role-based permissions

Several projects required different permissions for different user roles.

I implemented role flags, admin checks and backend rules to control access to sensitive data and management features.

### Production support

Admin panels were used not only for content management, but also for real production support: checking users, payments, tickets, orders and app data.
